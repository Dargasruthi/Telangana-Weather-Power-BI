# Telangana State Weather Data Analysis (Power BI)

## Project Objective
To analyze weather patterns of Telangana state using Power BI and extract meaningful insights.

## Dataset
- Telangana weather data (CSV)
- Daily weather records for multiple districts

## Tools Used
- Power BI Desktop
- GitHub

## Visualizations
- KPI Cards (Avg Temperature, Rainfall, Humidity)
- Line Chart (Temperature Trend)
- Bar Chart (Rainfall by District)
- Pie Chart (Weather Conditions)
- Slicers (District, Date)

## Key Insights
- Certain districts receive higher rainfall during monsoon
- Temperature peaks during summer months
- Humidity varies across regions

## Files
- `.pbix` – Power BI project file
- `.csv` – Dataset
